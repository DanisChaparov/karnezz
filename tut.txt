## 0. КАК УСТРОЕНА СРЕДА (CLUSTER)

### Папки:

- `Shared/` — **только чтение**  
  Там лежат:
  - датасеты задач (`train.csv`, `test.csv`, `items.parquet`, `sample_submission.csv`),
  - локальные модели (например, LLaMA),
  - вспомогательные файлы.
  > Ничего туда не записывать. Только копировать оттуда к себе.

- `Permanent/` — **моё рабочее место**  
  Всё важное делаю здесь:
  - код,
  - ноутбуки,
  - сохранённые модели,
  - `submission.csv`.

- `Work/` — временная папка (можно игнорить).

---

## 1. ОБЩИЙ ПЛАН РАБОТЫ В VS CODE

1. Открываю VS Code → открываю папку `Permanent/` как рабочую.
2. Создаю подпапку под задачу:
   - `Permanent/taskA/`
   - `Permanent/taskB/`
3. Копирую данные из `Shared` в `Permanent`:

   ```bash
   mkdir -p ~/Permanent/taskA
   cp /Shared/taskA/* ~/Permanent/taskA/

   mkdir -p ~/Permanent/taskB
   cp /Shared/taskB/* ~/Permanent/taskB/
